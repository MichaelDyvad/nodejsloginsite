Callback functions

A callback function is a function passed to another function as an argument. The way a callback functions operates is by being invoked after the first function completes.

The reason why we need Callback Functions in Javascript is because Javascript is asynchronous. So when a javascript program is running, other code can still run even though a function is not complete. A callback's function is therefore to execute code in reponse to an event.

app.get("/", (req, res) => {
  res.send(frontPageRender);
});

app.get("/login", (req, res) => {
    res.send(loginPageRender);
});

app.get("/signup", (req, res) => {
    res.send(signupPageRender);
});
Arrow Functions

Arrow functions were introduces in ES6 and makes it possible for us to write sorter functions

In arrow function there is a implicit return. Arrow function does not have their own "this" keyword and "this" does not change the value of car in the function.

The implicit return in arrowfunctions is great for making one-line operations for looping datastructures with loops such as map or filter

//arrow function vs. regular function and the use of this keyword
const car = "Volvo"
    let newCar = {
    car : "Ford",
    arrowFunction: () => {
        console.log(this.name); 
    },
    regularFunction() {
        console.log(this.name); 
    }   
}

newCar.arrowFunction(); // Volvo
newCar.regularFunction(); // Ford
Loops

Loop methods in javascript is:

map() : The map() method is for looping 1:1 and returns a new array. The map() method calls a function for each element in an array and will not change the original content

filter() : The filter() method filtrate the array on some perameter that we provide. The filter() methods creates a new array with the elements that passed the parameters we provided

reduce() : The reduce() methods purpose is to reduce the content in the data structure. The method executes a reducer function which returns the funtions accumulated result

forEach() : The forEach() methods iterates over every element in the given data structure.

for() : The vanilla for loop is only for finger counting

Fetching

Fetching in javascript is an important feature when it comes to API's. Fetching makes it possible to get json data on diffrent topics like pokemon.

Fetching gets the data from API's and we can use the reponse to tailor the data to our own needs. In the frontend fetching below we can get the correct img on a pokemon based on id. This is an example of how we can use the fetched data to accommodate our needs.

//Fetching back-end to make an api from an online api
  router.get("/api/pokemon", (req, res) => {
    fetch("https://pokeapi.co/api/v2/pokemon")
    .then(res => res.json())
    .then(data => {
        res.send({pokemon:data})
    })
  });
